Thank you for installing {{ .Chart.Name }}.

Your release is named {{ .Release.Name }}.

To verify that the pods are running, run:

  kubectl get pods -n {{ .Release.Namespace }} -l "release={{ .Release.Name }}"

To get the status of the services, run:

  kubectl get svc -n {{ .Release.Namespace }} -l "release={{ .Release.Name }}"

To access Prometheus:

  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-prometheus 9090:9090

Then visit http://localhost:9090

To access Grafana:

  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-grafana 3000:80

Then visit http://localhost:3000

Default Grafana credentials:
  Username: {{ .Values.grafana.adminUser }}
  Password: {{ .Values.grafana.adminPassword }}

To access Loki:

  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-loki 3100:3100

Then visit http://localhost:3100

To access Alertmanager:

  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-alertmanager 9093:9093

Then visit http://localhost:9093

  https://prometheus.io/docs/introduction/overview/
